# Uber Sales App
Uber　Driverの売上管理アプリです。

## 概要 
- 売上登録(Create)
- 売上一覧表示(Read)
- 売上編集(Update)
- 売上削除(Delete)
- 総売上表示
- 総配達件数表示
- 今月売上表示
- 今月平均時給表示
- 売上グラフ表示(Chart.js)
- 日付順ソート
- 売上順ソート


## 使用技術 
- HTML
- JavaScript
- Node.js
- Express
- SQLite
- Chart.js


## 今後追加予定 
- レスポンス対応
- 月別集計機能
- カレンダー表示
- UI改善


## 学習目的 
JavaScript Node.js SQLite　を用いた
CRUDアプリ開発の学習を目的として制作しました。


## 工夫した点 

- CRUD機能を実装し、売上データの登録　編集　削除ができるようにした
- 売上順と日付順で並び変えできるようにした
- Chart.jsを利用して売上推移をグラフで可視化できるようにした
- 配達件数や平均時給などを自動計算して表示するようにした


## 苦労した点 

- JavaScriptの非同期処理(fetch)の理解
- データ更新時の画面描画処理
- ソート機能実装時のデータ管理
- SQLiteとNode.jsの連携
- グラフ表示用データの加工
